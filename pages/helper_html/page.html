Plugin
  1. plugin.init
    Is executed when the module is loaded, can be used to change tray. plugin.init = (trayMenu, mainMenu) => { // do something }
  2. plugin.onPageLoad
    Is executed when page is done loading. It will receive a page class. plugin.onPageLoad = (page) => { // do something }
  3. plugin.beforePageClose
    Is executed when page is closing. It will receive a page class. plugin.beforePageClose = (page) => { // do something }
  4. plugin.onPageClose
    Is executed after the page closes. plugin.onPageClose = () => { // do something }
  5. plugin.beforeQuit
    Is executed when app is closing or module is reloading. plugin.beforeQuit = () => { // do something }
  6. plugin.log
    log at console. plugin.log('hi!');
Page
  1. page.addEventListener(event, query, function)
    Add a event to a element. page.addEventListener('click', '#btn-1', () => { plugin.log('button clicked!') });
    If the event is on a input may get the input value page.addEventListener('change', '#select-1', (value) => { plugin.log(`select has a value! ${value}`) });
  2. page.loadHtml(query, option)
    Load HTML to the parent specified on query page.loadHtml('#element', { clearBeforeRender: true, // can be ommited (default is true) fileOrHtml: string (absolute file path) | Array<HTMLObject> });

HTMLObject
HTMLObject is a object pattern to be trancribed to HTML elements.

It will be used in the followed instructions

{ tag: 'input', // required if isn't a role id: 'input-one', // is required for input elements text: 'hi' // shown as text inside of element class: 'class-one class-two' 'data-attribute': 'this is a data for "attribute"' children: Array<HTMLObject> } { role: 'modal-dismiss' }
  1. role not required, role may need some attributes.
    'Role' at this time cannot have custom HTML attributes.
  2. tag is required if role not specified.
  3. id in input is required if don't have a name.
  4. name in input is required if don't have a id.
  5. any other HTML attributes.
HTMLObject - Roles
Standard HTML element. 'Role' at this time cannot have custom HTML attributes.
{ role: 'modal-dismiss' }
  1. modal-dismiss create a button to close a modal with (origin 'event-close').
  2. form
    { role: 'form' children: Array<HTMLObject> }
  3. form-input
    { role: 'form-input' label: 'A text' // not required type: 'text' // if 'text' can be ommited name: 'input-1', // required col: 12 // not required, is a bootstrap collumn children: Array<HTMLObject> }
    Types: text (default), textarea, number, date, datetime, datetime-local, file
  4. form-select
    { role: 'form-select', label: 'A text', // not required name: 'select-1', // required col: 12, // not required, is a bootstrap collumn options: [ { label: 'one', value: '1' }, 'two' ] // can be empty array or undefined }
  5. table
    { role: 'table', id: 'select-1', // required if not inform NAME name: 'select-1', // required if not inform ID col: 12, // not required, is a bootstrap collumn caption: 'A table of something' // not required columns: Array<string> // required }

{ title: 'some title', // required body: Array<HTMLObject> | 'some text', // required footer: Array<HTMLObject> // required on: function }
