HTMLObject
HTMLObject is a object pattern to be trancribed to HTML elements.

It will be used in the followed instructions

{ tag: 'input', // required id: 'input-one', // is required for input elements class: 'class-one class-two' 'data-attribute': 'this is a data for "attribute"' children: Array<HTMLObject> } { role: 'modal-dismiss' }
  1. role not required, role may need some attributes.
    'Role' at this time cannot have custom HTML attributes.
  2. tag is required if role not specified.
  3. id in input is required if don't have a name.
  4. name in input is required if don't have a id.
  5. any other HTML attributes.
Modal
Modal is util for some popup action.
{ title: 'some title', // required body: Array<HTMLObject> | 'some text', // required footer: Array<HTMLObject> // required on: function }
  1. on Receive three parameters: event, origin, data.
    event -> is a object with a mothod: response. (should be added more latter).
    origin -> a name or id from which button the user clicked, is only for a footer.
    data -> if the modal has input they will be show in 'data'.
Roles
Standard HTML element. 'Role' at this time cannot have custom HTML attributes.
{ role: 'modal-dismiss' }
  1. modal-dismiss create a button to close a modal with (origin 'event-close').
  2. form
    { role: 'form' children: Array<HTMLObject> }
  3. form-input
    { role: 'form-input' label: 'A text' // not required type: 'text' // if 'text' can be ommited name: 'input-1', // required col: 12 // not required, is a bootstrap collumn children: Array<HTMLObject> }
    Types: text (default), textarea, number, date, datetime, datetime-local, file
  4. form-select
    { role: 'form-select', label: 'A text', // not required name: 'select-1', // required col: 12, // not required, is a bootstrap collumn options: [ { label: 'one', value: '1' }, 'two' ] // can be empty array or undefined }
