JavaScript

Plugin
  1. plugin.init
    Is executed when the module is loaded, can be used to change tray. plugin.init = (trayMenu, mainMenu) => { // do something }
  2. plugin.onPageLoad
    Is executed when page is done loading. It will receive a page class. plugin.onPageLoad = (page) => { // do something }
  3. plugin.beforePageClose
    Is executed when page is closing. It will receive a page class. plugin.beforePageClose = (page) => { // do something }
  4. plugin.onPageClose
    Is executed after the page closes. plugin.onPageClose = () => { // do something }
  5. plugin.beforeQuit
    Is executed when app is closing or module is reloading. plugin.beforeQuit = () => { // do something }
  6. plugin.log
    log at console. plugin.log('hi!');
Page
  1. page.addEventListener(event, query, function)
    Add a event to a element. page.addEventListener('click', '#btn-1', () => { plugin.log('button clicked!') }); If the event is on a input may get the input value page.addEventListener('change', '#select-1', (value) => { plugin.log(`select has a value! ${value}`) });
  2. page.loadHtml(query, option)
    Load HTML to the parent specified page.loadHtml('#element', { clearBeforeRender: true, // can be ommited (default is true) fileOrHtml: string (absolute file path) | Array<HtmlObject> });
